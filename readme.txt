# eBay-Preis

## Beschreibung

Mit der eBay-Preis Chrome-Erweiterung kann man schnell nach bereits verkauften Artikeln auf eBay suchen. Einfach den gewünschten Text markieren, rechtsklicken und "Auf eBay suchen" auswählen. Ein neuer Tab öffnet sich mit den Suchergebnissen auf eBay. Effizient und zeitsparend!

## Installation

1. Klone das Repository oder lade es herunter:
    ```bash
    git clone https://github.com/TobiasWatz/ebay-preis.git
    ```

2. Öffne Google Chrome und gehe zu `chrome://extensions/`.

3. Aktiviere den Entwicklermodus (Schalter oben rechts).

4. Klicke auf "Entpackte Erweiterung laden" und wähle den Ordner `ebay-preis` aus, den du gerade heruntergeladen oder geklont hast.

5. Die Erweiterung wird nun in Chrome installiert und ist einsatzbereit.

## Nutzung

1. Markiere einen Text auf einer Webseite.
2. Klicke mit der rechten Maustaste und wähle "Auf eBay suchen".
3. Ein neuer Tab öffnet sich und zeigt die Suchergebnisse auf eBay an.

## Haftungsausschluss

Diese Erweiterung stammt nicht von eBay und steht in keiner Verbindung zu eBay. Sie wurde unabhängig entwickelt und dient lediglich dazu, die Suche nach verkauften Artikeln auf eBay zu erleichtern.
